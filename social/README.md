# Social Media Prompts

> **Purpose:** Platform-optimized social media content

## Files

- **`formats/all-platforms.md`** - Complete guide for all social platforms
  - LinkedIn, Twitter/X, Instagram, Threads, Facebook
  - Platform specifications and character limits
  - Voice calibration by platform
  - Output templates

## Usage

**For all platforms:**
```
Use: social/formats/all-platforms.md
Output: Posts for LinkedIn, Twitter, Instagram, Threads, Facebook
```

**For single platform:**
Reference the specific section in `all-platforms.md` for that platform.

## Platform Specs

| Platform  | Character Limit | Hashtags | Best Format |
| --------- | --------------- | -------- | ----------- |
| LinkedIn  | 3,000 (700 visible) | 3-5 max | Story-driven, professional |
| Twitter/X | 280 (threads unlimited) | 1-2 max | Sharp hooks, threads |
| Instagram | 2,200 | 5-10 | Visual-first, emotional |
| Threads   | 500 | 3-5 max | Conversational, casual |
| Facebook  | 63,206 (80 visible) | 1-3 max | Community-focused |

## Voice by Platform

- **LinkedIn:** Founder-grade insight, strategic, credible
- **Twitter/X:** Sharp and direct, one idea per post
- **Instagram:** Warm, visual, aspirational
- **Threads:** Casual, conversational, real-time
- **Facebook:** Accessible, community-oriented
