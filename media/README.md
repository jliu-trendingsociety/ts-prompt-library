# Media Generation Prompts

> **Purpose:** AI image and video generation

## Files

### Images
- **`images/prompt-templates.md`** - AI image generation prompts
  - Platform specifications (aspect ratios, dimensions)
  - Visual brand guidelines
  - Prompt architecture
  - Tools: Midjourney, DALL-E, Flux, Ideogram, Stable Diffusion

### Video
- **`video/script-templates.md`** - Video script templates
  - Short-form content (TikTok, Reels, Shorts)
  - Long-form content (YouTube)
  - AI video prompts
  - Script structures

## Usage

**For images:**
```
Use: media/images/prompt-templates.md
Output: 2-3 image prompts with platform specs
```

**For video:**
```
Use: media/video/script-templates.md
Output: Video script or AI video prompt
```

## Platform Image Specs

| Platform              | Aspect Ratio | Dimensions             |
| --------------------- | ------------ | ---------------------- |
| Instagram Feed        | 1:1 or 4:5   | 1080x1080 or 1080x1350 |
| Instagram Story/Reels | 9:16         | 1080x1920              |
| LinkedIn              | 1.91:1 or 1:1 | 1200x627 or 1080x1080  |
| Twitter/X             | 16:9 or 1:1  | 1600x900 or 1080x1080  |
| Facebook              | 1.91:1       | 1200x630               |
| YouTube Thumbnail     | 16:9         | 1280x720               |
| Blog/OG Image         | 1.91:1       | 1200x630               |
