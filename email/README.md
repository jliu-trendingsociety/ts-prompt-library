# Email Marketing Prompts

> **Purpose:** Email sequences and newsletters

## Files

- **`sequences.md`** - Complete email marketing templates
  - Welcome sequences
  - Newsletter formats
  - Launch emails
  - Nurture flows

## Usage

**For newsletters:**
```
Use: email/sequences.md
Command: "email this"
Output: Newsletter format from source content
```

**For sequences:**
Reference specific sequence type in `sequences.md`.

## Email Types

- **Welcome Sequences** - Onboarding new subscribers
- **Newsletters** - Regular content distribution
- **Launch Emails** - Product/service announcements
- **Nurture Flows** - Educational content series
