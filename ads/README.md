# Paid Advertising Prompts

> **Purpose:** Ad copy for paid campaigns

## Files

- **`copy-formats.md`** - Ad copy templates
  - Meta (Facebook/Instagram) ads
  - Google ads
  - LinkedIn ads
  - YouTube pre-roll

## Usage

**For ads:**
```
Use: ads/copy-formats.md
Command: "ads"
Output: Meta + LinkedIn ad versions
```

## Platform Formats

- **Meta Ads** - Facebook/Instagram feed and stories
- **Google Ads** - Search and display
- **LinkedIn Ads** - Sponsored content
- **YouTube Ads** - Pre-roll and mid-roll
